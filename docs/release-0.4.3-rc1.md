# Ubuntu Hibernate Wizard 0.4.3 RC1

Release candidate created after real-system testing.

## Versioning

- User-visible app version: `0.4.3-rc1`
- Python package version: `0.4.3rc1`

Automated test suite must pass before publishing this release candidate. GTK smoke may be skipped in the sandbox when GTK/gi runtime is unavailable.
